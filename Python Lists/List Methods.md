## append() Method = appends an element to the end of the list.
```python
fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(fruits)
```
```bash
['apple', 'banana', 'cherry', 'orange']
```
## List copy() Method = returns a copy of the specified list.
```python
fruits = ["apple", "banana", "cherry"]
x = fruits.copy()
print(x)
```
```bash
['apple', 'banana', 'cherry']
```
